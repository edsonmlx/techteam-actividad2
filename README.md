## 🌿 Flujo de Trabajo con Git

### Ramas del proyecto

```bash
main                    # Rama principal (producción)
feature/ENCABEZADO      # Líder — Header y navegación
feature/CUERPO          # Colaborador 1 — Contenido principal
feature/PIE-PAGINA      # Footer e información de contacto
```

### Comandos utilizados

```bash
# Clonar el repositorio
git clone https://github.com/usuario/techteam-actividad2.git

# Crear y cambiar de rama
git checkout -b feature/ENCABEZADO

# Agregar cambios
git add .

# Realizar un commit
git commit -m "feat: agregar estructura del encabezado"

# Subir la rama al repositorio remoto
git push origin feature/ENCABEZADO

# Crear Pull Request en GitHub → esperar revisión del líder
```

## 🚀 Cómo ejecutar el proyecto

### Página web

Abre `index.html` directamente en tu navegador.

### Script Python

Requiere Python 3.6 o superior. No tiene dependencias externas.
